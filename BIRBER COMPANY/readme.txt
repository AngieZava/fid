Welcome to Birdsite!
The site contains general information about why the company was made. 
There are boxes with images of each type of bird, when a box is clicked, it takes the user to the page containing information on that specific bird.
The bird pages all provide links to recomended accomodations for the type of bird it is about. There is also a home button at the bottom that takes the user back to the homepage.
The site is a light blue color with buttons that grow when hovered over.
